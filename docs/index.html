About Zombs Arena

This is my capstone project for 15-113. It is a 2D PvPvE shooter game heavily inspired by zombsroyale.

Game Details: There are 4 weapon types: pistols, shotguns, ARs, and snipers. There are weapon rarities represented by 6 colors (Gray, Green, Blue, Purple, Gold, Red). Increasing rarity either increases the damage or changes the number of pellets (shotgun). Running is faster with fists equipped.
