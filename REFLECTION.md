1. Describe your process
I started with the base arena and player and slowly added things into the project to make it more like a game. I made some prompts very specific on what I wanted to add, but left some prompts more vague when I was unsure of how exactly I wanted to implement a feature. 

2. What AI tools and strategies did you use?
I used Gemini Pro in a browser chat, and mostly talked to it by describing what I wanted to add and how I wanted to add it.

3. Why did you make the choices above?
I didn't feel like my project idea was very complicated, so I didn't feel the need to use agentic development. I also wanted to see how the AI implemented features as I went, so constantly testing the project locally after each prompt/change was the most effective strategy for me.

4. What changed from your pre-113 approach?
I believe I've improved my prompting. Before, I wasn't really sure how to explain ideas to the AI. Now, I better understand how to word my ideas and when to let the AI try to decide on details on its own. I also feel like I better understand token limits, so I try to be more efficient with my prompts.

5. What might you do with this project with more time?
With more time, I would most likely make this game more flushed out. The graphics are currently not very good, but the core gameplay mechanics are there. There are some things missing from the original game, like structures and ammo mechanics. I would most likely focus on these with more time.
